# NestFi Documentation Site

Coming soon page for NestFi documentation at docs.nestfi.io.

## Getting Started

### Install Dependencies

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the page.

### Build

```bash
pnpm build
```

### Start Production Server

```bash
pnpm start
```

## Deployment

### Vercel (Recommended)

1. Push this directory to a separate repository or as a subdirectory
2. Import the project in Vercel
3. Set the domain to `docs.nestfi.io`
4. Deploy

### Manual Deployment

1. Run `pnpm build`
2. Deploy the `.next` folder to your hosting provider
3. Configure DNS to point `docs.nestfi.io` to your hosting

## Features

- ✅ NestFi theme matching main site
